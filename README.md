# 🛍️ Customer Segmentation using Unsupervised Machine Learning

## 📌 Project Overview
Customer Segmentation is an Unsupervised Machine Learning project that groups customers based on their purchasing behavior and demographic information. This project uses the Mall Customers dataset to identify distinct customer groups using the K-Means Clustering algorithm. These customer segments help businesses create targeted marketing strategies, improve customer satisfaction, and increase sales.

---

## 🎯 Objectives
- Analyze customer behavior using data visualization.
- Perform data preprocessing and feature selection.
- Apply the K-Means Clustering algorithm.
- Determine the optimal number of clusters using the Elbow Method.
- Visualize customer segments.
- Interpret each customer group for business insights.

---

## 📂 Dataset
**Dataset Name:** Mall Customers Dataset

### Features
- CustomerID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1-100)

## 🔄 Project Workflow

### 1. Import Libraries
Load all required Python libraries.

### 2. Load Dataset
Read the Mall Customers dataset into a Pandas DataFrame.

### 3. Exploratory Data Analysis (EDA)
- Check dataset information
- Check missing values
- Statistical summary
- Distribution plots
- Pair plots
- Correlation analysis

### 4. Data Preprocessing
- Select important features
- Scale numerical data using StandardScaler

### 5. Find Optimal Number of Clusters
Use the **Elbow Method** to determine the best value of **K**.

### 6. Train K-Means Model
Fit the K-Means clustering algorithm with the optimal number of clusters.

### 7. Predict Customer Clusters
Assign each customer to a cluster.

### 8. Visualization
Visualize customer groups using scatter plots.

### 9. Business Insights
Interpret each cluster based on spending score and annual income.

---

## 📊 Machine Learning Algorithm
### K-Means Clustering
K-Means is an unsupervised machine learning algorithm that partitions data into K clusters by minimizing the distance between data points and their respective cluster centroids.

## 📌 Conclusion
This project demonstrates how Unsupervised Machine Learning can uncover hidden customer patterns without labeled data. By using the K-Means Clustering algorithm, customers are grouped into meaningful segments based on income and spending behavior. These insights enable businesses to make data-driven decisions, improve marketing strategies, enhance customer engagement, and maximize overall business growth.
